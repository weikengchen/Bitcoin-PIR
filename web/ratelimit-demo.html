Anonymous Rate-Limiting Demo

ARC (multi-show credential) vs Cashu Blind Auth (single-show tokens), end-to-end: mint → obtain → present → verify.

This demo is self-contained — it talks only to the dev-issuer (free issuance, no Lightning, no PIR database). Start it first:
cargo run -p dev-issuer  then cd web && npm run dev.
The dev-issuer co-locates the verify gate (identical crypto to the PIR server's unified_server gate), so no PIR server is needed. In production the same present frames go over WebSocket to the PIR server.
